Smart Home Device Management
Problem Statement:
Design a smart home device control system:
Base class Device with deviceId, deviceName, status (on/off).
Subclasses: Light, Thermostat, SecurityCamera, each with unique features.
Interface Controllable with methods turnOn(), turnOff(), and getStatus().
Implement different device behaviors via interface.
Use encapsulation for device settings.
Support scheduling device operations.
Override methods to handle device-specific functionality.
Key OOP concepts:
Interfaces for device control.
Inheritance and polymorphism.
Encapsulation for device data.
